# GitHub commands

# Inicializar el repositorio local con el repositorio remoto
```
git init
```

- Referencia del repositorio local con el repositorio remoto (GITHUB)
```
git remote add origin https://github.com/G17hub-us3r/back_nest_angularAB.git
```

# Validar referencia
```
git remote -v
```
